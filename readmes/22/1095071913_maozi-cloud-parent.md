# maozi-cloud-parent

<br/>

基于 Spring-Cloud-Alibaba-Dubbo 开源封装一站式分布式解决方案，它包含开发分布式应用程序所需的所有组件，为开发团队代码风格一致，提升代码质量，专注高效快速开发业务

<br/>

亮点组件：全链路灰度 、全链路完整日志 、本地高效开发模式 、入参出参映射数据库条件查询 、分布式锁 、服务监控 、领域模型 、结构设计

<br/>

未开发组件：WebSocket（认证授权、ACK重试发送、兼容多实例）、方法失败重试、升级最新SCA、Oauth2替换最新SAS认证授权、ARM架构的本地灰度开发、AI规则训练开发
