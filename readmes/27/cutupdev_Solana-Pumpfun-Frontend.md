- Trading chart
- Other features


## How to install it

```bash
git clone https://github.com/cutupdev/Solana-Pumpfun-Frontend.git
```

```bash
cd ./Solana-Pumpfun-Frontend
```

```bash
npm run install
```

```bash
npm run dev
```
